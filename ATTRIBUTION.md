- **Fork Relationship:** This repository is an official fork maintained by [armpit-symphony](https://github.com/armpit-symphony)

## What Was Changed

SparkFish is a rebrand and continuation of MiroFish by Spark Pit Labs. Changes include:

- Full rebrand from MiroFish to SparkFish (name, logo, messaging)
- Removal of Shanda Group branding and references
- Removal of Chinese-language content from UI and documentation
- Translated system prompts from Chinese to English
- Infrastructure and dependency updates

Full commit history of changes is available in the git log.
